== Description ==

This is Pixer Widgets for WordPress by ThemePixer. With Pixer Widgets you can easily add your site details, Newsletter and Recent Posts with Widgets on Footer and also sidebar.
Here in Pixer Widgets you will find 
About Widgets,
Newsletter Widgets,
Recent Posts Widgets
 & MORE COMING.....




== Installation ==
1. Upload `pixerwidgets.zip` to the `/wp-content/plugins/` directory and exctract
2. Activate the plugin through the 'Plugins' menu in WordPress
